This application was designed and created by David Goodman for a University of California, Santa Cruz computer science (cmps101) class assignment. The project has been made open source! Visit us on GitHub.

This is an implementation of the Game of Life, which is a cellular automaton invented by Cambridge mathematician John Conway. It was written entirely in Java using Google Web Toolkit (GWT). Although the application is written in Java no Java Runtime Environment (JRE) is required to run it because GWT cross-compiles everything into JavaScript and optimizes and obfuscates it to be run on any web server and in any browser (including Androids and iPhones).

Rules for Life:

  1. Any live cell with fewer than two live neighbours dies, as if caused by under-population.
  2. Any live cell with two or three live neighbours lives on to the next generation.
  3. Any live cell with more than three live neighbours dies, as if by overcrowding.
  4. Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.
These rules govern the Game of Life at a fundamental level, but the most fascinating aspect of the game are the new rules and larger patterns that emerge over the turns. In fact, this game is a universal computer and creatures such as gliders can be utilized to represent all of the essential logic gates needed to build a computer.
